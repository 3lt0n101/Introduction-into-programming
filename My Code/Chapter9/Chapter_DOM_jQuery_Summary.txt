ðŸ”¹ What is the DOM?
DOM = Document Object Model.
It's the browser's tree-like structure of your HTML page.
JavaScript uses the DOM to read and change content, style, and structure.

ðŸ”¹ Accessing Elements:
document.getElementById("id")
document.getElementsByClassName("class")
document.querySelector("selector")

ðŸ”¹ Modifying Content:
let title = document.getElementById("title");
title.textContent = "New Title";

ðŸ”¹ Changing Styles:
title.style.color = "red";
title.style.fontSize = "24px";

ðŸ”¹ Event Listeners:
title.addEventListener("click", function() {
    alert("Title clicked!");
});

ðŸ”¹ Creating Elements:
let newPara = document.createElement("p");
newPara.textContent = "Hello!";
document.body.appendChild(newPara);

ðŸ”¹ What is jQuery?
jQuery is a library that simplifies DOM tasks and works well across browsers.

ðŸ”¹ Basic Syntax:
$(selector).action();

Example:

$("#box").css("background-color", "black");

ðŸ”¹ Handling Events:

    alert("Button clicked!");
});

ðŸ”¹ Effects and Animations:

ðŸ”¹ Modifying HTML:

ðŸ”¹ Adding/Removing Classes:

$("#box").removeClass("dark");
